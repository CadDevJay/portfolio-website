# Jaydeo Gavali Portfolio Website

Futuristic, engineering-focused personal portfolio designed for GitHub Pages.

## What This Includes

- Immersive home section with mechanical/AEC-inspired visual language
- Scroll-linked reveal effects and animated UI transitions
- Lightweight WebGL 3D background built with Three.js (CDN import)
- Complete one-page architecture with all required sections:
	- Home
	- Blogs
	- My Work / Projects
	- My Life
	- Clients I Have Worked With
	- Testimonials
	- Learn With Me
	- Free Learning Resources
	- Hobbies
	- Contact Me

## Project Structure

- `index.html`
- `assets/styles.css`
- `assets/app.js`

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy To GitHub Pages

1. Push this repository to GitHub.
2. Go to repository `Settings` -> `Pages`.
3. Under `Build and deployment`:
	 - Source: `Deploy from a branch`
	 - Branch: `main`
	 - Folder: `/ (root)`
4. Save and wait for deployment.
5. Your site will be available at:
	 - `https://<your-username>.github.io/portfolio-website/`
